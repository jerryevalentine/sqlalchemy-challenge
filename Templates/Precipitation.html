API - Precipitation Data

This API endpoint returns the precipitation data for the last 12 months.

Usage

To access the precipitation data, make a GET request to the following endpoint:

/api/v1.0/precipitation

Example Response

        {
            "2016-01-01": 0.08,
            "2016-01-02": 0.0,
            "2016-01-03": 0.0,
            ...
        }
